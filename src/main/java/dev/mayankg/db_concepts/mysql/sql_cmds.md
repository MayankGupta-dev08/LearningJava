### Show Databases
```sql
SHOW DATABASES;
```

### Create Database
```sql
CREATE DATABASE <database_name>;
```

### Drop Database
```sql
DROP DATABASE <database_name>;
```

### Use Database
```sql
USE <database_name>;
```
